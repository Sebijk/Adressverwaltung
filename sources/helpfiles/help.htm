Sebijks Adressverwaltung - Hilfe zu Befehlen Sebijks Adressverwaltung: Hilfe zu Befehlen

Hier finden Sie Hilfe bei den einzelnen Befehlen:

Auflisten
Listet die Adressen auf.

Adressen eingeben

Hiermit können Sie neue Adressen eingeben.

Suchen

Wenn Sie viele Adressen eingetragen haben, und eine bestimmte Suchen,
können Sie Suchfunktion verwenden.

Alle Adressen löschen

Löscht alle Adressen. Diese Funktion ist mit einem Passwort geschützt, die Sie unter
"Info" finden.

Drucken

Druckt die Adressliste aus. Bevor Sie die Adressliste drucken, stellen Sie sicher, dass sie
Adressen aufgelistet haben. Es ist empfehlenswert im Querformat zu drucken.

Neu laden

Lädt das Programm neu.

Alle Adressen sichern

Sichert die Adressen.

Alle Adressen wiederherstellen

Stellt alle Adressen die sie gesichert haben wieder her. Diese Funktion ist mit einem
Passwort geschützt, die Sie unter "Info" finden.

Info

Zeigt das Dialogfenster Info an.

Hilfe

Zeigt die Hilfe von Sebijks Adressverwaltung an.
